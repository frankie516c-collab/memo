```
//반복문 기본 문법
//1. 전역 변수
for(/* 2. 지역 변수 */ ; ; ) {
    console.log("반복");
}
```
- 1번은 밖에서도 가능
  - `var` 또는 `let`을 사용할 수 있음
  - `let`을 사용하는 것을 추천
      - why? 변하는 수면 안되고 정해진 값이어야 하기 때문
      - e.g.,
        ```
        let i = 0;
        ```
- 2번은 안에서만 사용 가능

```
let i = 0;
for( ; i < 1 ; ) {
    console.log("반복");
    i = 1;
}
console.log(i);
```
- 실행: `반복` 값 `1`
- why? i가 1보다 작을 때만 true이기 때문에 반복이 되는데 `i = 1`이라는걸 넣어졌으니까 1번 돌고 멈추게 될 것

```
let i = 0;
for( ; i < 1 ; ) {
    console.log("반복");
    i = i + 1;
}
console.log(i);
```
- `i = i + 1;`이게 뭔가 **증가시킬 영역**이기 때문에 세번째 콜롬에 넣어도 됨

```
let i = 0;
for( ; i < 1 ;  i = i + 1 ) {
    console.log("반복");
}
console.log(i);
```
**- 이렇게 for에 썼을 때 반복되는 공식이 더 쉽게 눈에 띄기 때문에 이렇게 쓰는 것을 추천한다**
